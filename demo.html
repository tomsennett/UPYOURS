UPME.js Guide and Demo

Learn how to use UPME.js for interactive web elements with Pico.css for styling.

Introduction to UPME.js

UPME.js is a lightweight JavaScript library that enhances web pages with interactive functionality. By adding the UPME class to an element along with a specific functionality class, UPME.js automatically assigns the corresponding JavaScript function to that element.

How to Include UPME.js in Your Project

To use UPME.js, include the script in your HTML file. Here's an example of how to do it:

<script src="path/to/upme.js"></script>

Interactive Demos

Hover Color Change

This example changes the background color of the element when hovered over.

Hover over me!
<div class="UPME hoverColorChange" style="padding: 1em; border: 1px solid #ccc;">Hover over me!</div>

<script>
function UPME_hoverColorChange() {
    if (this.event_mouseover) {
        this.style.backgroundColor = '#ffc107';
    }
    if (this.event_mouseout) {
        this.style.backgroundColor = '';
    }
}
</script>

Live Character Count

Counts the number of characters typed into a text area and displays the count live.

Characters: 0

<textarea class="UPME liveCharCount" placeholder="Start typing..."></textarea>
<p class="charCountDisplay">Characters: 0</p>

<script>
function UPME_liveCharCount() {
    if (this.event_input) {
        const count = this.value.length;
        document.querySelector('.charCountDisplay').textContent = `Characters: ${count}`;
    }
}
</script>

Toggle Visibility

A button that toggles the visibility of a block of text.

This text can be toggled.

<button class="UPME toggleVisibilityBtn">Show/Hide Text</button>
<p class="toggleText">This text can be toggled.</p>

<script>
function UPME_toggleVisibilityBtn() {
    if (this.event_click) {
        const textBlock = document.querySelector('.toggleText');
        if (textBlock) {
            textBlock.style.display = textBlock.style.display === 'none' ? 'block' : 'none';
        }
    }
}
</script>

Real-Time List Filter

Filters a list of items in real-time based on the input value.

  • Apple
  • Banana
  • Cherry
  • Date
  • Elderberry
<input type="text" class="UPME listFilterInput" placeholder="Filter list...">
<ul class="itemList">
    <li>Apple</li>
    <li>Banana</li>
    <li>Cherry</li>
    <li>Date</li>
    <li>Elderberry</li>
</ul>

<script>
function UPME_listFilterInput() {
    if (this.event_input) {
        const filterValue = this.value.toLowerCase();
        const items = document.querySelectorAll('.itemList li');
        items.forEach(item => {
            const text = item.textContent.toLowerCase();
            item.style.display = text.includes(filterValue) ? '' : 'none';
        });
    }
}
</script>

Dynamic Content Loading

A button that loads additional content into the page dynamically.

<button class="UPME loadContentBtn">Load More Content</button>
<div class="contentArea"></div>

<script>
function UPME_loadContentBtn() {
    if (this.event_click) {
        const contentArea = document.querySelector('.contentArea');
        if (contentArea) {
            const newContent = document.createElement('p');
            newContent.textContent = 'This is dynamically loaded content!';
            contentArea.appendChild(newContent);
        }
    }
}
</script>

Conclusion

UPME.js offers a straightforward way to add interactivity to your web pages. By using the UPME class and corresponding functionality classes, you can easily implement dynamic behaviors. Try these examples and explore more possibilities with UPME.js!
